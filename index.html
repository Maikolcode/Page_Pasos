Pasos para el desarrollo de la pagina

Recursos multimedia

- Crear una carpeta donde contenga todas las imagenes que se van a utilizar dentro de la pagina.

- Identificar la gama de colores y tipo de font que se implementara en la pagina.

Lenguajes y consumo de API's

- Acordar tecnologias que se utilizaran en el codigo para una correcta maquetación, en este caso se haria uso HTML, CSS y JS para la maquetación con ayuda del framekor Bootstrap.

- Para utilizar el buscador de direcciones sera necesario consumir la api de Google Maps, para lograr con exito el uso de este recurso es necesario que se cree una cuenta en Google Cloud Platform.

- Estando en la consola de Google Cloud se deben habilitar las siguientes API's: Maps JavaScript Api y Places API.

- Despues de habilitar las API´s se debe generar una Key la cual sera utilizada en el documento html para el consumo de dicho servicio.

Diseño de la interfaz y logica del negocio

- Cuando se haga la carga del script sera momento de iniciar con el codigo de la interfaz, principalmente se usara flex-box, grid y bootstrap para un correcto posicionamiento de los elementos, ademas de ser responsive.

- A la hora de realizar el nav-bar o cabecera de la pagina sera necesario dejar el background-img = url('image_url') para poder poner elementos encima del fondo.

- En dicho proceso se agregara un input, este sera el encargado de consumir el servicio de autocomplete, su id estara conectada a un metodo llamado activarBuscador que comenzara a ejecutarse cuando el usuario decida realizar una busqueda.

- Para que nuestro nav-bar siga el sroll sera necesario cambiar un atributo de su clase a position fixed, de esta manera la cabecera estara fija en la parte superior de la pagina.

- En cuanto a las tarjetas de los pasos a seguir para el proceso de venta sera necesario el uso de Flex-box, esto se debe a que acomoda cada tarjeta de forma horizontal sin sobrepasar el width predeterminado como se muestra en el ejemplo:
